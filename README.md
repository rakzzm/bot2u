# bot2u

A native Android app for Text-to-Speech with voice cloning capabilities.

## Repository

https://github.com/rakzzm/bot2u

## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/rakzzm/bot2u.git
cd bot2u
```

2. Open the `android` folder in Android Studio

3. Build and run on an emulator or device

## Features

- Voice sample upload for voice cloning
- Text-to-speech synthesis
- Adjustable generation parameters
- Audio playback
- Modern Material 3 design

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Retrofit
- MVVM Architecture
